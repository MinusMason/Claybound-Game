\acknowledgements{
This section is used to acknowledge whoever's support and contribution.
The command that introduces it is ignored in the project literature review and progress report. It is used in the final report,  but  is not compulsory. If you do not
have an acknowledgements command in your preamble then there
won't be any acknowledgement section in the document produced. \emph{Abstract} and \emph{Acknowledgements} sections should fit on the same page.

\subsection{Original Objectives}
\label{app:obj}
\begin{enumerate}
    \item Project Proposal (Weeks 1-2): Draft a project proposal, receive feedback and update accordingly.
    
    \item Literature review (Weeks 2-6): Write up a literature review on the project comparing different methodologies and literature. Decide on what methodology to use.
    
    \item Design (Week 6-8): Look over the core algorithms and chosen method to deepen understanding
    
    \item Coding (Week 8-22): Code and implement the fluid simulation
    
    \item Testing and evaluation (8-26): Generally testing as code base develops and once done run evaluation testing
    
    \item Final report and project submission (Week 26-34): Write up final report and submit project
\end{enumerate}
